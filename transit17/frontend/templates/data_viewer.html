MTA Data Viewer

How to use

To view data, please run the following command:

python scripts/show_data.py

This will display all data statistics in the database.

Data collection

To start collecting data, please run:

python scripts/run_collector.py

This will start collecting real-time data from the MTA API.

Database management

To initialize the database, please run:

python scripts/init_db.py

To clear the database, please run:

python scripts/clear_db.py
